# CrudHub_With_Servlet

## 📌 Project Overview
This project is a dynamic **CRUD (Create, Read, Update, Delete) web application** developed using **Java Servlet, JSP, and MySQL**. The application allows users to manage records efficiently through a simple and user-friendly interface. It also includes **secure user authentication with login and logout functionality**.

## 🚀 Features
- 🔐 Secure **User Login & Logout Authentication**
- ➕ **Create** new records
- 📖 **Read/View** existing records
- ✏️ **Update** record details
- ❌ **Delete** records
- 🛡️ **Duplicate record prevention** to maintain data integrity
- 💾 MySQL **database integration**
- ⚡ **Session management for secure access**
- 📱 Responsive UI using **HTML, CSS.**

## 🛠️ Technologies Used
- **Java**
- **Servlet**
- **JSP**
- **JDBC**
- **MySQL**
- **HTML**
- **CSS**
